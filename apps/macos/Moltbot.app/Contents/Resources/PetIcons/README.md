# Pet Icons Directory

Put your pet state icons here (80x80 pixels, PNG format with transparent background):

## Required Files:
- `cat-offline.png`  - Disconnected from Gateway (gray theme)
- `cat-online.png`   - Connected and working (green theme)
- `cat-idle.png`     - Connected but waiting (blue theme)
- `cat-busy.png`     - Working on tasks (orange theme)

## Specifications:
- Size: 80x80 pixels (can also include @2x 160x160 for Retina)
- Format: PNG with transparent background
- Style: Cartoon, cute calico cat with blue knitted hat

## Example Filename:
```
apps/macos/Resources/PetIcons/cat-online.png
```

## Design Tips:
- Keep the calico cat pattern (black, orange, white)
- Include the blue knitted hat with yellow decoration
- Match colors to the state theme
- Express different emotions for each state

## Midjourney Prompts (for reference):
```

Sad/lonely expression, gray color theme, minimalist, flat design.
```
